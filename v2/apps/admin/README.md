# apps/admin

Admin console application.

Recommended stack:

- Next.js
- React
- Tailwind CSS
- shadcn/ui
- Radix
- TanStack Table
- React Hook Form
- Zod

Primary modules:

- Dashboard
- Media Library
- Album Studio
- AList Import Center
- Processing Jobs
- Storage Center
- Site Settings

The admin should feel like a lightweight image studio, not a database CRUD panel.
