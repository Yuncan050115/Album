# apps/web

Public gallery application.

Recommended stack:

- Astro
- React Islands
- Tailwind CSS
- GSAP
- Three.js / React Three Fiber where needed

Primary responsibilities:

- Home page
- Album pages
- Photo detail pages
- Tag archive pages
- SEO
- RSS
- Public image delivery

This app should stay visually advanced but technically light. Avoid turning the public gallery into a heavy SPA.
